Variáveis e Operadores
Tipos
Variáveis são utilizadas para que o compilador utilize a memória e armazene um quantidade específica dessa memória para uso posterior. No geral, é preciso declarar a variável e atribuir à ela algum valor. Em C, isso é feito da seguinte forma:
int x = 9;
Primeiramente, é preciso declarar o tipo da variável, para que o computador saiba a quantidade de memória que será alocada. Após a declaração, a variável recebe um nome, geralmente composto por letras do alfabeto. Futuramente, veremos como nomear variáveis de forma eficiente, por enquanto só é necessário saber que elas devem conter somente letras.
Após nomear a variável, é necessário atribuir-lhe valor. Não necessariamente isso é feito de imediato, ou seja, logo após nomear a variável. Há casos em que ocorre o seguinte:
int x;
x = 9;
De qualquer forma, se há a criação de uma variável, é porque queremos que ela possua um valor. Essa atribuição é feita colocando-se um símbolo de igual, bem semelhante à forma matemática. Vamos verificar se o valor de x é realmente 9:
int x = 9;
printf("&d",x)
Output: 9
Tipos de Variáveis
Em C existem vários tipos de variáveis, mas nos concentraremos por enquanto somente em dois tipos: int e float. Como vimos acima, o tipo int oferece à variável a característica de número inteiro. Com isso, podemos realizar operações matemáticas simples, como soma, subtração, multiplicação e divisão.
O tipo float é bem semelhante ao tipo int, com 2 diferenças primordiais: espaço ocupado na memória e uso em operações matemáticas.
