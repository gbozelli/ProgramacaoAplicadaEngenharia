Introdução à Programação
O que é a programação?
Programação é o processo de construir um programa para um computador. Os programas são constituídos por um conjunto de instruções que indicam o que o computador deve fazer para gerar um resultado esperado. Para escrever um programa, é preciso utilizar uma linguagem de programação. A linguagem faz papel de intermediário entre o computador e o programador, e possui uma sintaxe própria.
Linguagem de Baixo Nível, Médio e Alto Nível
Uma linguagem de baixo nível é a linguagem que o computador entende. Por ser uma representação direta de um código da máquina, é muito difícil compreendê-la diretamete e, por conta disso, foram criadas as linguagens de médio e alto nível. Uma linguagem de alto nível possui símbolos que são compreendidos por quem escreve o código, mas esses símbolos não podem ser compreendidos pelo computador, por isso precisam de uma espécie de "tradutor", que lê o código oferecido pela linguagem de alto nível e o traduz em uma linguagem de baixo nível. A linguagem de médio nível seria uma espécie de meio termo entre o baixo nível e o médio nível. É onde está o C.
O que é 'C'?
C é uma linguagem de programação criada por Ken Thompson. Inicialmente foi desenvolvida para o sistema operacional UNIX, e foi amplamente utilizada durante os anos 80 e 90 em diversas aplicações. Apesar de C, atualmente, não ser mais tão popular como antigamente, sua influência em outras linguagens de programação (como C++, Java e C#) o tornam essencial para a programação em geral, e suas aplicações em áreas da engenharia, como sistemas embarcados, ainda são muito grandes para ser ignorada.
Por ser uma linguagem de médio nível, um código escrito em C precisa ser convertido em linguagem de máquina para ser executado. Quem faz esse papel é o compilador. Utilizaremos o OnlineGDB como compilador, por ser de fácil acesso e extremamente prático.
