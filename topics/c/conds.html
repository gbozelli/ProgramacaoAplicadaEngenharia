Tomada de decisão
Ao resolver um problema, você se depara com diversas condições. Se essas condições forem verdadeiras ou não, o resultado do seu problema se altera, como se houvesse uma espécie de dependência entre a condição e o resultado. Vamos supor que você precisa separar, legalmente, adultos de não adultos, e, pela Constituição Federal, uma pessoa deve ter mais de 18 anos para ser considerada como um adulto.
Nesse exemplo, a condição é a pessoa ter mais de 18 anos para ser adulta, e nossos resultados são dois: a pessoa ser ou não ser, ou seja, nosso resultado ser falso or verdadeiro. Vamos tentar traduzir isso em um pseudocódigo:
Idade = 19
Se Idade > 18,
  é maior de Idade
Se não,
  é menor
Agora, vamos tentar aplicar isso usando as ferramentas de C
Comandos 'if' e 'else'
Na tradução livre, 'if' e 'else' significam, respectivamente, 'se' e 'se não'. Tem a seguinte estrutura:
if (condição) {
  // instruções
} // fim da condição
A estrutura nos diz que se a condição for verdadeira, todas as linhas de código delimitada pelas chaves serão lidas. Do contrário, o compilador simplesemente vai pular as instruções das chaves, e nada vai ser processado. Podemos incrementar o 'if' utilizando o 'else', que será lido quando a condição do 'if' for falsa:
if (condição) {
  // instruções
} else { //negação da condição
  //instruções }
Podemos também utilizar vários if quando temos mais de uma condição a ser testada, utilizando o camando else if:
if (condição 1) {
// instruções
} else if (condição 2) {
//instruções }.
.
.
} else if (condição n) {
  //instruções }
Vamos implementar o nosso exemplo, agora usando C:
int idade = 19;
if (idade > 18) {
  printf("É maior de idade");
} else {
  printf("Não é maior de idade");
}
Operadores lógicos e relacionais
Para relacionar duas variáveis, usamos operadores relacionais. Isso quer dizer que a condição a ser analisada na nossa tomada de decisão é a relação entre valores, e se essa relação é verdadeira ou não. Vamos observar alguns operadores relacionais:
Maior: >
Menor: <
Igual: ==
Menor e igual: <=
Maior e igual: >=
Utilizamos operadores lógicos para juntar condições ou invertê-las, ou seja, teremos uma ou mais condições que formarão outra. Por exemplo: Idade = 19
Se Idade > 18 E Idade < 60,
  é adulto
Nesse caso, Idade > 18 e Idade < 60 são duas condições unidas em uma só pelo operador lógico E, se tornando uma só condição Idade > 18 E Idade < 60. Outro exemplo:
Idade = 19
Se NÃO Idade > 18,
  é criança
O operador NÃO inverte nossa condição, tornando, nesse exemplo, a condição Idade > 18 igual à Idade <= 18 .
Não: !!
E: &&
Ou: ||
