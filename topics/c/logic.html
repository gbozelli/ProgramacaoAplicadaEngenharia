Lógica de Programação
Conceitos básicos
Lógica de programação é conexão de pensamentos para alcançar um objetivo através de um programa de computador. Os pensamentos são uma sequência de instruções, e as instruções indicam ao computador a sua tarefa.
Pensar em lógica de programação nos leva em direção à algoritmos. Algoritmos são uma sequência de passos que realiza uma tarefa, e esses passos devem ser claros e precisos (como uma receita de bolo). Escrever um programa nada mais é do que escrever um algoritmo de acordo com a sintaxe da linguagem de programação.
Escrevendo algoritmos
Para escrever um algoritmo, utiliza-se um pseudocódigo. Pseudocódigos são compostos de instruções genéricas, e são uma espécie de "rascunho" para o programa que será escrito posteriormente, mas não precisam conter os formalismos e regras da linguagem de programação.
A escrita do algoritmo é feita pensando-se em três etapas: a entrada, o processamento e a saída. Entrada são os dados que serão enviados ao computador, o processamento é o conjunto de procedimentos para atingir o resultado desejado e a saída são os dados já processados. A seguir, um exemplo básico de algoritmo que calcula a nota média de um aluno:
Entrada: P1, P2 e P3 (Notas das provas)
Processamento: Média = (P1+P2+P3)/3 (Cálculo da média)
Saída: Média
