Estruturas de Repetição
Comandos "while" e "for"
Muitas vezes nos vemos programando linhas e linhas, as quais, muitas vezes poderiam ser simplificadas em linhas de código menores, como por exemplo, quando queremos imprimir números de 0 a 9. Da seguinte forma:
printf("0");
printf("1");
printf("2");
printf("3");
printf("4");
printf("5");
printf("6");
printf("7");
printf("8");
printf("9");
Saída: 123456789
Está claro que perderíamos muito tempo e linhas repetindo os comandos, deixando nosso código maior e pouco intuitivo, e, desse modo, podemos utilizar uma estrutura de repetição para simplificar nossos códigos. Nesta seção, abordaremos as duas principais, while e for.
Comando "while"
Semelhante ao nosso if, a estrutura while é executada sob uma condição, e enquanto ela for atendida (ou seja, for verdadeira) o código continuará executando o laço de repetição.
int x = 0;
while(x < 10){ #Condição
printf("%d", x);
x = x + 1; //Soma um valor a x a cada vez que o laço é executado, equivale a x = x + 1
}

Saída:
0123456789
Comando "for"
Diferente do while, no for possuímos uma variável de controle, ou seja, uma variável que controla a execução do for. Essa variável é sujeita à uma condição, assim como no while, e de um acréscimo. Vejamos um exemplo:
for(int i = 0; i < 10; i = i + 1){
printf("%d", i);
}

Saída:
0123456789
A variável i é a nossa variável de controle, e, enquanto ela for menor que 10 (nossa condição) a linha de comando dentro das chaves será rodada, e, a cada execução, seu valor aumentará em uma unidade.
