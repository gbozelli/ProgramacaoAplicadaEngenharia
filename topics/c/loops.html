Estruturas de repetição
Comandos while e for
Muitas vezes nos vemos programando linhas e linhas, as quais, muitas vezes podem ser simplificadas em linhas de código menores, como por exemplo, quando queremos utilizar solicitar valores para o usuário, mas não queremos repetir o código. Da seguinte forma:
printf("Digite dez números: ");
int n1, n2, n3, n4, n5, n6, n7, n8, n9, n10 = 0;
printf("1o valor: ");
scanf("%i", &n1);
printf("2o valor: ");
scanf("%i", &n2);
...
Logo, percebemos que perderíamos muito tempo e linhas repetindo os comandos para cada entrada e saída. Desse modo, podemos utilizar uma estrutura de repetição para simplificar nossos códigos. Nesta seção, abordaremos as duas principais, while e for.
Comando while
A estrutura while é executada sob uma condição, e enquanto ela não for atendida o código continuará executando o laço de repetição.
int x = 0;
while(x < 10){ #Condição
printf("\n%d", x);
x++;
}
