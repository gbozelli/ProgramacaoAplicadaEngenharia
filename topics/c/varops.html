Variáveis
Variáveis são utilizadas para que o compilador utilize a memória e armazene um quantidade específica dessa memória para uso. No geral, é preciso declarar a variável e atribuir à ela algum valor, assim como fazemos em matemática . Em C, isso é feito da seguinte forma:
int x = 9;
Primeiramente, é preciso declarar o tipo da variável, para que o computador saiba a quantidade de memória que será alocada, e, após a declaração, a variável recebe um nome, geralmente composto por letras do alfabeto. Futuramente, veremos como nomear variáveis de forma eficiente, por enquanto é necessário saber que elas devem conter somente letras.
Após nomear a variável, é necessário atribuir-lhe valor. Não necessariamente a atribuição de valor é feita na mesma linha da declaração do tipo da variável. Em casos onde nós queremos descobrir o valor de x, só escrevemos o trecho da seguinte forma:
int x;
De qualquer forma, se há a criação de uma variável, é porque queremos que ela "guarde" alguma coisa, porque eventualmente usaremos essa "coisa". Essa atribuição é feita colocando-se um símbolo de igual, bem semelhante à forma matemática. Vamos verificar se o símbolo = fez seu trabalho:
int x = 9;
printf("%d",x);
Saída: 9
Tipos de variáveis
Em C existem vários tipos de variáveis, mas nos concentraremos por enquanto somente em dois tipos: int e float. Como vimos acima, o tipo int oferece à variável a característica de número inteiro, e, com isso, podemos realizar operações matemáticas simples, como soma, subtração, multiplicação e divisão.
O tipo float é bem semelhante ao tipo int, com 2 diferenças primordiais: espaço ocupado na memória e uso em operações matemáticas. Sobre a memória, o assunto será abordado com mais detalhes em tópicos mais avançados. Já sobre as operações matemáticas, o seu uso é recorrente em situações nas quais precisamos representar números decimais, como por exemplo, em operações financeiras, representações de números não naturais, etc.
Declaramos a variável float da mesma forma que declaramos a variável int:
float x = 9;
printf("%d",x);
Saída: 9.0
Operadores
Conhecendo as variáveis e sabendo utilizá-las, podemos partir para a manipulação das mesmas, sendo possível realizar operações matemáticas fundamentais, tais como adição, subtração, divisão e multiplicação.
int x = 1;
int y = 2;
int z = 0;

z = x + y; #Adição
printf("%d", z);
Saída: 3

z = x - y; #Subtração
printf("%d", z);
Saída: -1

z = x * y; #Multiplicação
printf("%d", z);
Saída: 2

z = x / y; #Divisão
printf("%d", z);
Saída: 0
Note que na divisão, o resultado impresso foi 0, apesar de sabermos que o resultado correto é 0,5, a linguagem interpretou a operação entre dois inteiros e retornou um valor inteiro, no caso o 0. Para este caso e outros, que envolvam maior precisão, o uso da variável float é recomendável. Assim, teríamos o seguinte:
float x = 1;
float y = 2;
float z = 0;

z = x + y; #Adição
printf("%f", z);
Saída: 3.0

z = x - y; #Subtração
printf("%f", z);
Saída: -1.0

z = x * y; #Multiplicação
printf("%f", z);
Saída: 2.0

z = x / y; #Divisão
printf("%f", z);
Saída: 0.5
Atente-se à sintaxe. Para variáveis do tipo int imprimimos/chamamos utilizando "%d" ou "%i", e para variáveis do tipo float utilizamos "%f". Caso queira representar as duas primeiras casas decimais, utilize "%.2f", caso queira as três "%.3f", e assim por diante.
