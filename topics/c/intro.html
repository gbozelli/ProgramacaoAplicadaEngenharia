Introdução à Programação
O que é a programação?
Programação é o processo de construir um programa para um computador. Os programas são constituídos por um conjunto de instruções que indicam o que o computador deve fazer para gerar um resultado esperado. Para escrever um programa, é preciso utilizar uma linguagem de programação, pois linguagem faz papel de intermediário entre o computador e o programador, como se fosse uma espécie de "tradutor".
Linguagem de Baixo Nível e Alto Nível
Uma linguagem de baixo nível é a linguagem que o computador entende. Por ser uma representação direta de um código da máquina, é muito difícil compreendê-la diretamete e, por conta disso, foram criadas as linguagens de alto nível. Uma linguagem de alto nível consegue ser compreendida por um humano, mas possui uma limitação: para o computador entendê-lá, precisa traduzir um código escrito em alto nível para um código em baixo nível.
O que é 'C'?
C é uma linguagem de programação criada por Ken Thompson, e, inicialmente foi desenvolvida para o sistema operacional UNIX. Apesar de C, atualmente, não ser mais tão popular como antigamente, sua influência em outras linguagens de programação (como C++, Java e C#) o tornam essencial para a programação em geral, e suas aplicações em áreas da engenharia, como sistemas embarcados, ainda são muito grandes para ser ignorada.
Por ser uma linguagem de alto nível, um código escrito em C precisa ser convertido em linguagem de máquina para ser executado e quem faz esse papel é o compilador. Inicialmente, não vamos nos preocupar em compreender esses conceitos de forma profunda, apenas entender o que eles fazem. Nesse curso, utilizaremos o OnlineGDB como compilador, por ser de fácil acesso e extremamente prático.
Lógica de programação
Lógica de programação é conexão de pensamentos para alcançar um objetivo através de um programa de computador. Os pensamentos se tornam uma sequência de instruções, que indicam ao computador a tarefa a ser realizada.
Pensar em lógica de programação nos leva em direção aos algoritmos, que é uma forma mais sofisticada de classificar nossos programas. Escrever um programa nada mais é do que escrever um algoritmo de acordo com uma linguagem de programação.
Parece complicado, mas vamos fazer um paralelo: Uma receita de bolo é um algoritmo, pois possui uma sequência de instruções que utilizam ingredientes para apresentar um resultado (o bolo).
Escrevendo algoritmos
Para escrever um algoritmo, podemos utilizar um pseudocódigo. Pseudocódigos são compostos de instruções genéricas e servem como uma espécie de "rascunho" para o programa que será escrito posteriormente, mas não precisam conter os formalismos e regras da linguagem de programação. Nós podemos escrever o algoritmo diretamente no código da linguagem, mas isso pode se tornar muito confuso no início, principalmente quando não temos uma noção muito grande de algoritmos.
A escrita do algoritmo é feita pensando-se em três etapas: a entrada, o processamento e a saída: Entrada são os dados que serão enviados ao computador, o processamento é o conjunto de instruções e a saída são os resultados. A seguir, um exemplo básico de pseudocódigo de um algoritmo que calcula a nota média de um aluno em uma matéria:
Entrada: P1, P2 e P3 (Notas das provas)
Processamento: Média = (P1+P2+P3)/3 (Cálculo da média)
Saída: Média
Como exemplo, o mesmo código escrito acima, porém em C (Não se preocupe em compreender o código, é apenas uma demonstração de como a definição do algoritmo antes da escrita do código facilita a implementação do mesmo):
int main(){
  int Nota_P1 = 6;
  int Nota_P2 = 8;
  int Nota_P1 = 7;
  int Media = (Nota_P1 + Nota_P2 + Nota_P3)/3;
  printf("&d",Media)
}
Output: 7
