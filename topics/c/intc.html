Programando em C
Estrutura de um programa
Um programa em C, comumente, terá a seguinte estrutura:
//conjunto de instruções

int main(){
  //declarações
}
Entende-se por conjunto de instruções como pedaços de códigos que fazem coisas que o C, sozinho, não faz. Basicamente, vamos importar códigos para realizar determinadas tarefas, e essas tarefas vão facilitar nossa escrita do código.
A biblioteca básica de C é a stdio.h (do inglês "Standard Input/Output"). Importamos essa biblioteca para usar as funções printf() e scanf(), que são configurações de "entrada" e "saída" (lembre da entrada e saída vista no tópico anterior). Não se preocupe com seu uso e implementação agora, ambas serão explicadas futuramente.
O trecho int main() inicialemnte pode parecer bastante confuso. Ele declara uma função (a função executa um conjunto de instruções) int main, de "principal". Essa função é definida dentro da linguagem, e deve ser utilizada para que o código seja executado. Se essa função não for colocada, o compilador não lê o código, e não há execução.
E as declarações? São as instruções da receita de bolo. Posteriormente, veremos que declarações podem ser colocadas fora do escopo da função principal. Por ora, elas serão colocadas aqui dentro, e incluem declarações de variáveis (lembre de variáveis em matemática), estruturas de repetição (loops), condicionais, operadores lógicos, entre outras.
Vamos entender a funcionalidade do seguinte código. Antes, execute-o no através do OnlineGDB:
#include <stdio.h>

int main(){
  printf("Estou estudando C!");
  return 0;
}
