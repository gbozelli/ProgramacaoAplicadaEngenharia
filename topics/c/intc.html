Programando em C
Estrutura de um programa
Um programa em C, comumente, terá a seguinte estrutura:
//conjunto de instruções

int main(){
  //declarações
}
Entende-se por conjunto de instruções como pedaços de códigos que possuem funcionalidades de entrada e saída não disponíveis na linguagem nativa do C. Basicamente, vamos importar códigos para realizar determinadas tarefas.
