Vetores e Matrizes
Armazenando valores
Até agora, armazenamos valores utilizando variáveis, o que é completamente comum e aceitável em contextos mais práticos, como armazenar o raio de uma circunferência, a idade de uma pessoa, entre outras. Mas e se tivessemos que amrmazenar 100 valores distintos? Criaríamos 100 variáveis? Isso seria pouco prático e poderia deixar nosso programa mais confuso. Para isso, usamos vetores: int vetor[tamanho] = {itens};
Um vetor possui: tipo de dado a ser armazenado ( como int ou float), tamanho (quantidade de 'variávies') e os itens (valor de cada variável). Vejamos um exemplo:
//Usando vetores:
int vetor[3] = {1,2,3};
//Usando variáveis:
int x = 1;
int y = 2;
int z = 3;
Utilizar vetores acaba sendo mais fácil e prático do que usar variáveis, mas a principal vantagem está na manipulação. Aqui o nível de dificuldade crescerá um pouco, pois utilizaremos loops com vetores para demonstrar como essa ferramenta é poderosa, já que os vetores nos introduzem a noção de estrutura de dados. Vamos supor que você quer armazenar os números de 1 a 9 em um programa, e existem duas maneiras de fazer isso:
//Usando variáveis:
int a = 1;
int b = 2;
int c = 3;
int d = 4;
int e = 5;
int f = 6;
int g = 7;
int h = 8;
int i = 9;
//Usando vetores:
int vetor[9]; for (int i = 0; i < 9 ;i += 1)
vetor[i] = i+1;
Quando um vetor está sendo utilizado, é preciso chamar a posição que se deseja do vetor para acessar seu valor. No exemplo acima, cada índice do vetor recebeu um valor próprio, que é o seu próprio índice somado com uma unidade. O que fizemos aqui com loops foi, basicamente, percorrer um vetor, ou seja, o loop é a forma básica de percorrer um vetor por completo.

//imprimindo os valores de uma matriz for (int i = 0; i < m; i += 1) for (int j = 0; i < n; i += 1) printf("%d",matriz[i][j]);
